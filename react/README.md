# Reflex-Aware Chat Engine (React Demo)

## Run

```bash
cd react
npm install
npm run dev
```

## Display Prefix Toggle

- Default (raw tags):

```bash
npm run dev
```

- Converted display prefix (example `XX`):

```bash
VITE_DISPLAY_PREFIX=XX npm run dev
```

This only changes rendered tag prefix in the UI. Scalar parsing and dataset source tags remain unchanged.
